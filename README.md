# Integrated Automotive Service and Management System (IASMS)

This repository represents a capstone project focused on designing an integrated system for managing automotive services, customer data, inventory, and business operations.

## Project Overview
The system combines multiple components into one platform:
- ERP system (inventory, employees, services)
- E-commerce features (appointment booking, customer interaction)
- Data analytics and reporting
- Mobile accessibility
- Cybersecurity protections

## Purpose
This repository demonstrates the structure and design of the system, including a basic prototype file and system concept.

## Technologies
- React (frontend concept)
- Node.js (backend concept)
- MySQL (database concept)
